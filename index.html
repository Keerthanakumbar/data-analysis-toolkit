User Guide

Data Analysis Toolkit · Tools and Methods of Data Analysis (K_mcs_003)

SRH University of Applied Sciences Heidelberg · Leipzig Campus · 2026

infoWhat is this Toolkit?

The Data Analysis Toolkit is a fully browser-based, interactive statistical analysis platform built with HTML5, JavaScript, and Tailwind CSS. It requires no installation, no coding knowledge, and no internet connection after loading. Users can upload their own CSV or JSON datasets and immediately perform data exploration, statistical analysis, visualization, probability calculations, and hypothesis testing — all in one place.

play_circleHow to Use — Step by Step

1

Load a Dataset

Click "Upload CSV / JSON" on the Dashboard or drag and drop your file. A built-in sample dataset with 10 subjects (Trial_Value, Group, Control) is available if you have no file ready.

2

Explore the Data

Navigate to "Data Exploration" to view the raw data table, check for missing values, filter rows, and export a cleaned CSV report.

3

View Descriptive Statistics

Go to "Descriptive Statistics" to see mean, median, standard deviation, variance, min, max, and range — automatically calculated from your dataset.

4

Visualize

Open "Visualization" to generate an interactive histogram showing the frequency distribution of values across defined ranges.

5

Probability Distributions

Use the "Probability" module to compute PDF, CDF (P(X≤x), P(X>x)), Z-scores, and percentiles via the Normal Distribution Calculator. A live normal curve chart updates as you type.

6

Statistical Inference

Run a One-Sample T-Test, compute Confidence Intervals, compare two groups, and run the Shapiro-Wilk Normality Test — all with real statistical output and pass/fail interpretation.

datasetSample Dataset

The built-in demo dataset simulates a clinical trial with 10 subjects. It contains three columns:

ColumnTypeDescription
SubjectTextSubject identifier (e.g. Alpha_22, Gamma_44)
Trial_ValueNumberMeasured numeric value for analysis (range: ~8–24)
ControlBooleanTRUE = control group, FALSE = treatment group
